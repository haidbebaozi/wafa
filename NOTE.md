# 项目开发
## 项目初始化
* 1. 环境搭建
`npm install -g @vue/cli`
* 2. 创建项目
`vue create wafa`
* 3. 启动
```
cd wafa
npm run serve
```

## 上传github
```
git remote add origin https://github.com/haidebaozi/wafa.git

git checkout -b wafa

git status

git add .

git commit -m "初始化"

git push

git push --set-upstream origin wafa
```
